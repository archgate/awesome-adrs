# prettier-oxlint

Code formatting with Prettier and linting with Oxlint for TypeScript and JavaScript projects.

## Included ADRs

| ID       | Title                         |
| -------- | ----------------------------- |
| ARCH-001 | Code Formatting with Prettier |
| ARCH-002 | Code Linting with Oxlint      |

## Quick Start

Import the full pack:

```bash
archgate adr import packs/prettier-oxlint
```

Cherry-pick a single ADR:

```bash
archgate adr import packs/prettier-oxlint/adrs/ARCH-001-formatting-prettier
```
