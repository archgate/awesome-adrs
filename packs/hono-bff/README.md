# hono-bff

Backend for Frontend with Hono — BFF pattern, type-safe API with Zod OpenAPI, and standardized folder structure.

## Included ADRs

| ID     | Title                                       |
| ------ | ------------------------------------------- |
| BE-001 | Backend for Frontend with Hono              |
| BE-002 | Type-Safe API with Hono RPC and Zod OpenAPI |
| BE-003 | Backend Folder Structure                    |

## Quick Start

Import the full pack:

```bash
archgate adr import packs/hono-bff
```

Cherry-pick a single ADR:

```bash
archgate adr import packs/hono-bff/adrs/BE-001-bff-hono
```
