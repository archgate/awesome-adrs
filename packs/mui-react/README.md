# mui-react

MUI (Material UI) as the React component library — theming, layout primitives, and accessible components.

## Included ADRs

| ID     | Title                        |
| ------ | ---------------------------- |
| FE-001 | MUI as UI Component Library  |

## Quick Start

Import the full pack:

```bash
archgate adr import packs/mui-react
```

Cherry-pick a single ADR:

```bash
archgate adr import packs/mui-react/adrs/FE-001-mui-component-library
```
