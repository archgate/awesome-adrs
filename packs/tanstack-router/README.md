# tanstack-router

File-based routing with TanStack Router — type-safe routes, params, search params, and loaders.

## Included ADRs

| ID     | Title                                  |
| ------ | -------------------------------------- |
| FE-001 | TanStack Router for File-Based Routing |

## Quick Start

Import the full pack:

```bash
archgate adr import packs/tanstack-router
```

Cherry-pick a single ADR:

```bash
archgate adr import packs/tanstack-router/adrs/FE-001-tanstack-router
```
