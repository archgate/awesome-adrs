# moonrepo

Monorepo management with Moonrepo — task orchestration, caching, and standardized folder structure.

## Included ADRs

| ID        | Title                                  |
| --------- | -------------------------------------- |
| ARCH-001  | Monorepo with Moonrepo                 |
| ARCH-002  | Standardized Monorepo Folder Structure |
| ARCH-003  | Centralized Dependency Versions        |

## Quick Start

Import the full pack:

```bash
archgate adr import packs/moonrepo
```

Cherry-pick a single ADR:

```bash
archgate adr import packs/moonrepo/adrs/ARCH-001-monorepo-moonrepo
```
