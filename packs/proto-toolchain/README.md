# proto-toolchain

Toolchain version management with Proto — pin and synchronize tool versions from a single file.

## Included ADRs

| ID        | Title                                  |
| --------- | -------------------------------------- |
| ARCH-001  | Toolchain Version Management with Proto |

## Quick Start

Import the full pack:

```bash
archgate adr import packs/proto-toolchain
```

Cherry-pick a single ADR:

```bash
archgate adr import packs/proto-toolchain/adrs/ARCH-001-toolchain-proto
```
