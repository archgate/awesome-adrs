# bun-runtime

Bun as the unified package manager and TypeScript runtime — replaces npm/yarn/pnpm and ts-node/nodemon.

## Included ADRs

| ID        | Title                              |
| --------- | ---------------------------------- |
| ARCH-001  | Bun as Package Manager and Runtime |

## Quick Start

Import the full pack:

```bash
archgate adr import packs/bun-runtime
```

Cherry-pick a single ADR:

```bash
archgate adr import packs/bun-runtime/adrs/ARCH-001-bun-package-manager-runtime
```
