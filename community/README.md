# Community Links

This directory contains a curated index of external ADR repositories that are compatible with Archgate.

Community links are pointers to third-party repositories — they are not reviewed to the same depth as curated packs under `packs/`, but they must be archgate-compatible and maintained.

## How it works

The `links.yaml` file lists external repositories that contain archgate-compatible ADRs. You can browse these links to discover ADR collections maintained by the community.

## Adding a link

See [CONTRIBUTING.md](../CONTRIBUTING.md) for submission guidelines, or use the [submit community link](https://github.com/archgate/awesome-adrs/issues/new?template=submit-community-link.yml) issue template.
